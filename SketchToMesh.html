Sketch-To-Mesh

Sketch-To-Mesh is the project I am currently working on for my final at Full Sail.

Sketch-To-Mesh is a Blender add-on that allows a user to input a picture and get a model in return.

We mostly wrote in python and we used libraries like bcyrpt, tensorFlow and open-cv

There are also functionalities for a database and we worked with a bit of machine learning.


During this project I've worked with Jira for the first time.

I think the role I aligned with most during our project was project manager.

I create most of the tasks, I helped others with there tasks and I also created some of the main functions for the project.

The biggest addition to the project I made was creating the code to make hte model.

My code served as the backbone for this part of the project and with some help from my teammates we got this
