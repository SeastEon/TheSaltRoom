TheBasement

TheBasement is an app that is supposed to store junk information

It was built using Android Studios, but instead of using Java I used kotlin

I orginally, made theBasement because I was inspired by a file I have on my computer of the same name

TheBasment text document was just a place where I stored random information that I may or may not ever use.

TheBasement app was supposed to replicate this function, but it would save your information encyrpted to a database.

This meant no memory had to be used to while using the program.


I was actually amazed to load up the project and have it still run.

From my experience working as a solo developer for TheBasement for 2 Months I learned how to develop changes quickly.

From this experiecen I learnt what it means to be adaptable. Since we have a time crunch I could spend too long on spefic features
